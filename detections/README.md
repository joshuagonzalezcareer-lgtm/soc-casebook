# Detections

Custom detection content built and validated in the lab.

| Folder | Contents |
|--------|----------|
| `sigma/` | Portable SIGMA detection rules |
| `wazuh/` | Wazuh-native rules (local_rules.xml snippets) |
| `yara/`  | YARA malware signatures |

Each rule should note: what it detects, the data source, and the MITRE ATT&CK technique.
